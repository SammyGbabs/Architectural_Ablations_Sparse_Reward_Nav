# Figures

All figures used in the paper, the slides, and internal reports live here. **Every figure in this directory must be regenerable from a script under `Analysis/`.** No ad-hoc Jupyter cell outputs, no `final_v3_actually_final.png`.

## Naming convention

`{phase}_{what}_{detail}.png`

- `phase` ∈ `{p1, p2, p3, p4, intro, methods}`. Use `intro` for figures shown in the introduction (e.g., env render) and `methods` for diagrams (e.g., architecture sketches).
- `what` is what is being plotted (e.g., `iqm`, `learning_curve`, `capacity_heatmap`, `trajectory`).
- `detail` disambiguates within `what` (e.g., `dqn_exp5`, `ppo_inverted_vs_symmetric`).

**Good:** `p1_iqm_dqn_all_configs.png`, `p2_capacity_heatmap_return.png`, `p3_lipschitz_distribution.png`, `intro_env_render.png`.

**Bad:** `plot.png`, `figure1.png`, `final.png`, `samuel_v2.png`.

## Rules

1. **One generating script per figure.** The script lives in `Analysis/` and its filename mirrors the figure: `Analysis/plot_p1_iqm_dqn_all_configs.py` → `figures/p1_iqm_dqn_all_configs.png`. Scripts that produce multiple related figures are fine; just list them all in the manifest.
2. **Every script reads from W&B or `results/csv/`,** never from a hard-coded number embedded in the script. If the underlying data changes, re-running the script regenerates the figure.
3. **Every script writes to `figures/` (this directory),** not to a Notebook output cell.
4. **Add an entry to the manifest below** the same PR you commit the figure in. No manifest entry → CI rejects the PR (once we set CI up).
5. **Plot style is consistent:** seaborn defaults, the project palette (`tab10` for categorical, `viridis` for sequential), Arial font, axis labels with units, legible at 1× scale.
6. **Output at 300 DPI** for anything that might go in the paper. For internal slides, 150 DPI is fine; mark these as `_slide` in the filename so they don't get mixed up.
7. **No PDFs in this directory.** PDFs of figures get generated at paper-compile time by the LaTeX repo, not stored here. PNGs only.

## Regenerating everything

```bash
# From repo root:
python -m Analysis.regenerate_all_figures
```
